Ici nous présenterons le langage de programmation Javascript. Javascript est langage de programmation crée en 1995 par Brendan Eich de chez Netscape Communications Corporation. C’est un langage de programmation particulièrement dynamique et principalement faite pour les des applications web. Il a été conçu pour rendre les sites web interactifs à la place d’être de simple texte brut avec un peu de style mis en avant. Grâce à ceci les sites web obtiennent en quelque sorte « le mouvement » avec le html qui sert de squelette et le ccs comme stylisera.

Bien que le nom puisse prêter confusion, ce langage n’a absolument rien à voir avec le langage de programmation Java.

Voici un exemple de code Javascript qui permet d’afficher Hello world sur une Canvas d’un fichier Html :

var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
ctx.fillStyle = "#000000";
ctx.font = "30px Arial";
ctx.fillText("Hello, world!", 10, 50);

Ceci permet d’afficher uniquement Hello world sur un Canvas avec la couleur noire (#000000)

Il peut être lis de cette façon :

Au début nous récupérons le Canvas du document html qui comporte l’ID myCanvas et on le stock en tant qu’objet avec le var et on lui donne le nom de Canvas.

Ensuite on déclare que ce Canvas doit être utilisé comme un Canvas 2d, si on veut faire un environnement 3d il faudra à la place écrire var ctx = canvas.getContext("webgl"); et on stocke le tout en tant qu'objet.

Nous déclarons ensuite une couleur de remplissage pour les caractères contenus dans la Canvas avec la ligne ctx.fillStyle = "#000000"; Ici il fera du noir.

Puis comme vous pouvez le deviner, la ligne ctx.font = "30px Arial"; permet de déclarer que le style du texte qui doit être utilisé est du Arial élargi à 30 pixels.

Puis finalement, la dernière ligne ctx.fillText("Hello, world!", 10, 50); permet d’imprimer le texte sur le Canvas qu'on déclaré ou début
