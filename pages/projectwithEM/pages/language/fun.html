Les languages de programmation explicite

Certains lagunages de programmation sont créés pour le fun uniquement tels que Malboge qui est conçu spécifiquement pour être le langage de programmation le plus dur au monde et le plus incompréhensible. Il a été créé en 1998 par Ben Olmstead et comme mentionner précédemment il n’a aucun autre but que d’être dur. Voici un exemple de code qui permet d’imprimer sur une console le mot « Hello world » :

(=<`#9]~6ZY327Uv4-QsqpMn&+Ij"'E%e{Ab~w=_:]Kw%o44Uqp0/Q?xNvL:`H%c#DD2^WV>gY;dts76qKJImZkj

Même pour les programmeurs expérimentés, ce bout de code est très difficile à lire et à comprendre vu que la syntaxe de code est faite de manière cryptique et complexe. Tout de même nous allons essayer de l’expliquer. Ce programme commence par une parenthèse pour montrer juste que le code commence, ensuite il déclare des variables avec les caractères, =, < et `. Ces variables seront ensuite utilisées durant le reste du code qui est beaucoup trop complexe pour être expliqué, mais en résumé, elle fait une sorte « d’équations » qui calculera le code de chaque caractère de « Hello world » et l’imprimera à tour de rôle.

Il existe sinon plusieurs autres langages de programmation du même type tels que le « BrainFuck » dont le même programme ressemblera à ceci :

++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.

Ce langage fonctionne exactement comme la machine que Turing a conçue. Avec le « + » qui ajoute une valeur à un certain endroit, le « - » qui recémente la valeur totale de 1, les « . » qui permette d’imprimer les caractères en fonction du contenu dans chaque endroit et les « [] »qui permette de faire des boucles. Les « <, > » permette de naviguer dans les endroits du ruban, ou plutôt de la mémoire.

Ou encore le langage de programmation « Chef » dont le même code est écrit comme une liste pour créer une recette exactement comme en cuisine :
